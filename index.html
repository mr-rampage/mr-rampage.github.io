DADOS 3.0 Checkpoint

Three Major Goals

  1. Data Integrity - data must be an accurate representation of the truth
  2. Stability - service and data must always be available
  3. Flexibility - system can support growth due to ever changing business needs

DADOS as a Platform

DADOS 3.0 will be designed as a platform constructed with two different tools:

  • Electronic Data Capture System - This component's goal is to allow the user to enter data into DADOS for storage.
  • Reporting System - This component's goal is to provide the user with analysis about the data stored within DADOS.

Both systems will appear under the same UI as DADOS 3.0, but from an architecture point of view will have separate applications and databases.

Application State

The design goal of DADOS 3.0 is to create a simple state machine which uses modules that follow a simple workflow. For future development, workflow can be defined per module allowing for increased flexibility and complexity.
